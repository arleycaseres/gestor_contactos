# 📒 Gestor de Contactos

Aplicación de consola para gestionar contactos personales, construida en Python.

## ¿Qué hace?
- Agregar contactos con nombre, teléfono y email
- Buscar un contacto por nombre
- Listar todos los contactos
- Eliminar contactos

## Tecnologías
- Python 3
- Estructuras de datos: diccionarios

## Cómo ejecutarlo
```bash
python3 contactos.py
```

## Aprendizajes
- Funciones en Python
- Diccionarios y métodos (.get, .items, del)
- Manejo de input del usuario
- Control de flujo con while y match
